Git es un sistema de control de versiones distribuidas de código abierto y gratuito diseñado para manejar todo, desde proyectos pequeños a muy grandes, con velocidad y eficiencia. Ramificación y fusión Al suscribirse (haga clic aquí ), puede asegurarse de no perderse las discusiones de seguimiento y también puede aprender sobre otros desarrollos en la comunidad. El archivo de la lista se puede encontrar en la bandeja de entrada pública.
este es un repositorio de solo publicación y todas las solicitudes de extracción se ignoran. Los errores en git se pueden reportar directamente a la lista de correo (ver arriba para más detalles). Tenga en cuenta que no necesita suscribirse a la lista para enviarla. Puede ayudarnos tratando de reproducir el error en la versión más reciente de git, o si está dispuesto a crear git desde la fuente, la nextrama .
